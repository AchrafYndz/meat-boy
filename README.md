## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- SFML
- CMake

### Installation

1. Clone the repo
```sh 
   git clone git@github.com:AchrafYndz/meat-boy.git
```
2. Build the project
```sh
   mkdir build
   cd build
   cmake ..
   make
```

### Run the game

Run the game from the meat-boy directory
```sh
  cd ..
  ./build/Game
```
