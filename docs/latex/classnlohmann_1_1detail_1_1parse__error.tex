\doxysubsection{Detailed Description}
exception indicating a parse error 

This exception is thrown by the library when a parse error occurs. Parse errors can occur during the deserialization of JSON text, CBOR, Message\+Pack, as well as when using JSON Patch.

Member {\itshape byte} holds the byte index of the last read character in the input file.
